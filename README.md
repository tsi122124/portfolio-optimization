# 📈 Portfolio Management Optimization Using Time Series Forecasting

## Project Overview

This project was completed as part of the 10 Academy Artificial Intelligence Mastery Program.

The objective is to develop an end-to-end financial analytics pipeline that leverages historical market data and forecasting techniques to optimize investment portfolio allocation.

The project integrates:

- Financial data collection using YFinance
- Exploratory Data Analysis (EDA)
- Statistical time series forecasting (ARIMA/SARIMA)
- Deep Learning forecasting (LSTM)
- Modern Portfolio Theory (MPT)
- Portfolio Optimization
- Strategy Backtesting

---

## Business Problem

Guide Me in Finance (GMF) Investments seeks to improve portfolio management by incorporating predictive analytics into investment decision-making.

Rather than relying solely on historical averages, analysts will forecast future market behavior and incorporate those forecasts into portfolio optimization while balancing expected return and investment risk.

---

## Project Objectives

- Collect historical financial data from Yahoo Finance
- Explore and understand financial market behavior
- Forecast Tesla stock prices using multiple forecasting models
- Compare statistical and deep learning approaches
- Construct an optimal investment portfolio
- Validate portfolio performance using historical backtesting
